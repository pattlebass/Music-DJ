# MusicDJ Redux

A modern recreation of the iconic MusicDJ app found on Sony Ericsson devices of the 2000s.

![MusicDJ Redux showcase](https://github.com/user-attachments/assets/504da973-2910-4b53-bf36-f3935908a5f5)

## Features
* Create songs using the original samples
* Open original MusicDJ MIDI files
* Export songs as WAV files
* Choose between Dark and Light themes, with more planned

## Roadmap
* High quality samples
* Adjustable tempo
* Faster exporting
* CLI
